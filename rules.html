Règles du jeu

  • Contrôlez votre ninja pour naviguer à travers les niveaux.
  • Éliminez les ennemis et évitez les obstacles pour atteindre la ligne d'arrivée dans le temps imparti.
  • Gagnez des points d'expérience pour améliorer votre ninja.
  • Les trois meilleurs temps de chaque mois recevront un NFT unique.
  • Les NFTs peuvent être échangés sur le marché.
